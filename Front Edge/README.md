# Front-Edge Academy

Site estático/interativo com HTML, Tailwind CSS (CDN), CSS custom e JavaScript puro.

## Scripts
- `npm run dev`
- `npm run build`
- `npm run preview`

## Estrutura
- `index.html`: estrutura principal da página
- `styles.css`: tema dark, cards, bordas e responsividade
- `script.js`: filtros, busca, menu mobile e Mini IDE

Sem React, sem login e sem área autenticada.
