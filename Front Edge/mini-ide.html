Mini IDE de treino orientado

Use a IDE para testar hipóteses rápidas, validar exercício de módulo e criar protótipos antes de levar para o projeto principal.

Fluxo recomendado

1) Leia o objetivo do módulo. 2) Implemente mínimo viável. 3) Rode. 4) Refatore. 5) Compare com checklist.

Checklist de qualidade

Semântica, responsividade, estados de erro, legibilidade de código e separação de responsabilidades.

Quando usar

Validação de componente isolado, teste de API mockada e experimentos de UX sem quebrar projeto principal.
